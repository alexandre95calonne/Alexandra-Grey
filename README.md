# ✂️ Alexandra Grey — Luxury Hair Salon Website

Website created for Alexandra Grey, a renowned hairdresser with four premium salons in the Paris region.  
Goal: elevate the brand’s digital presence with an elegant platform and robust admin tools.

💎 Luxe visuals, e-gift card system, blog & event management, and MapBox integration.  
🛠️ Tech stack: **Next.js**, **Tailwind CSS**, **Next UI**, **Supabase**, **MapBox**
